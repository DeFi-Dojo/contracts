### renounceOwnership

```solidity
function renounceOwnership() external nonpayable
```



*Leaves the contract without owner. It will not be possible to call `onlyOwner` functions anymore. Can only be called by the current owner. NOTE: Renouncing ownership will leave the contract without an owner, thereby removing any functionality that is only available to the owner.*


### setProxy

```solidity
function setProxy(address _address, address _proxyForAddress) external nonpayable
```

Allow the owner to set a proxy for testing



#### Parameters

| Name | Type | Description |
|---|---|---|
| _address | address | The address that the proxy will act on behalf of |
| _proxyForAddress | address | The proxy that will act on behalf of the address |
